Frequently asked questions

How does DocFlow Buddy detect when my docs are out of date?

We analyze your routes/controllers and compare them to your OpenAPI or markdown docs on each sync. When we see new, changed, or removed endpoints, we create a checklist and (optionally) fail CI or post a PR comment.

Do you modify my code or OpenAPI files automatically?

By default, no. DocFlow Buddy is designed to be safe and reversible: we surface diffs, generate suggestions, and let you apply them as commits or PRs from your own account.

Which languages and frameworks do you support?

We start with popular stacks solo founders use: FastAPI, Express/Node, Rails, and Laravel. You can still track generic HTTP services by pointing us at your OpenAPI spec.

Can I use this with private repos?

Yes. We connect via GitHub/GitLab tokens with least-privilege scopes and only store the metadata we need to detect doc drift.
